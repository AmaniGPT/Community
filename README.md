# Overview

AmaniGPT is an experimental LLM architecture **designed to work efficiently on commodity CPU clusters in order to reduce the need for powerful but expensive GPUs**.

This is the community repo where you can:
- File bug reports, make feature requests, and ask questions.
- Read tutorials and guides on how to use AmaniGPT.
